Expense Report Scraper

jQuery • Static HTML → JSON
This page uses jQuery selectors to scrape a static HTML expense-report snippet and extract structured JSON data — no server, no API. Click Parse Expenses to run the selectors against the rendered HTML on the left. The resulting JSON is compatible with the Finance Tracker's import format.

jQuery Selectors Used
Selector / Expression What It Extracts
$('.report-student').text() Student name from the report header
$('[data-student-id]').data('student-id') Student ID stored in a custom data attribute
$('.report-period').text() Reporting period string
$('.expense-row').each(function() { ... }) Iterates every expense row in the table
$row.data('ref') Reference number from the row's data-ref attribute
$row.find('.date').text() Date cell content per row
$row.find('.desc').text() Description cell content per row
parseFloat($row.find('.amount').text().replace(/[$,]/g, '')) Numeric amount — strips $ and , before parsing
$row.find('.cat-pill').text() Category label from the styled pill span
expenses.reduce((s, e) => s + e.amount, 0) Total amount summed from extracted rows
